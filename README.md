# Harp Homebrew Tap

This tap is for [Harp](https://github.com/elastic/harp) products.

## How do I install these formula

Install the tap via:

    brew tap elastic/harp

Then you can install individual products via:

    brew install elastic/harp/harp

The following products are supported:

* Harp `brew install elastic/harp/harp`
* Harp Server Plugin `brew install elastic/harp/harp-server`

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://github.com/Homebrew/brew/blob/master/docs/README.md).
